Initialization APIs

System initialization involves the following steps:

  • Identifying the devices in the system.
  • Creating a connection to each device.
  • Initializing the device to the point where software may control it.
  • Attaching an identifier to the device so that OpenNSL calls may refer to the device.

The first three steps are carried out by layers of software below the OpenNSL layer.

The OpenNSL API provide the functions to attach the device and then to initialize the software resources necessary to control the device and to initialize the device itself. The order of these operations is critical.
